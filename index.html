Overview

New Tabs Always Open Right of Current opens every new tab immediately to the right of the current tab including tabs opened from the toolbar new-tab button, keyboard shortcuts, page links, repeated child tabs, and links from other apps.

Web Store Availability

Chrome Web Store link coming soon.

The extension is built for Chrome, Brave, Edge, and other Chromium browsers that support Manifest V3 extensions from the Chrome Web Store.

Privacy Policy

This Privacy Policy applies to the New Tabs Always Open Right of Current browser extension.

Data processed by the extension

The extension processes tab and window metadata locally so it can place a new tab beside the tab that created it or beside the last active tab. This includes tab IDs, window IDs, active-tab state, opener/source tab IDs, pinned state, and tab indexes.

The extension does not store browsing history or page URLs. It uses source and target tab IDs from browser tab/navigation events only to decide where a newly opened tab should be moved.

Data stored by the extension

The extension stores short-lived session state in browser-managed extension storage. This state is used only to remember recent active/source tab relationships while the browser session is running.

What the extension does not collect

  • No names, email addresses, account identifiers, payment details, health information, location, or personal communications.
  • No page contents, form contents, passwords, cookies, or authentication tokens.
  • No analytics, advertising identifiers, telemetry, or tracking data.

Network requests and sharing

The extension does not send data to the developer, to a backend server, or to third-party analytics or advertising services. The developer does not sell user data.

The use of information received from Google APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Retention and deletion

Session state remains in browser-managed extension storage until cleared by the browser, replaced by newer state, or removed when the extension is uninstalled.
